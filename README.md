# 🏘️ Dalali App - Tanzania's Smart Property Marketplace

Dalali App is a cutting-edge real estate platform designed to revolutionize how people find, rent, and manage properties in Tanzania. Built with a **Mobile-First** approach and a focus on **Trust and Transparency**, Dalali App connects tenants directly with verified landlords and agents via an interactive map-based experience.

![Dalali App Preview](https://images.unsplash.com/photo-1560518883-93699c894819?auto=format&fit=crop&q=80&w=1200)

## 🚀 Key Features

*   **📍 Smart Map Discovery:** Real-time map-based search with interactive pins across major Tanzanian cities.
*   **📂 Dense Property Catalog:** Over 180+ unique listings across categories like **Single Rooms, Master Rooms, Full Houses, Apartments, Compounds (Plots), and Ukumbi (Halls)**.
*   **🎥 Rich Media Support:** High-quality image galleries (15+ photos per item) and full **Video Walkthrough** support for a remote-first viewing experience.
*   **🔍 Smart Street Selection:** Advanced "Post Property" logic that allows agents to select neighborhoods (Ubungo, Masaki, Sinza, etc.) and automatically syncs GPS coordinates.
*   **📱 Responsive Web Design (RWD):** Fully optimized for mobile, tablet, and desktop with a fluid Glassmorphism UI.
*   **🛡️ Escrow Protected:** Secure interest payment system to protect both tenants and landlords.
*   **🌍 Multi-Language & Dark Mode:** Support for English/Swahili and a premium Slate Dark Mode.

## 🛠️ Technology Stack

*   **Frontend:** React 18, TypeScript, Vite
*   **Styling:** Modern CSS3 (Flexbox & Grid), Glassmorphism, Animations
*   **Maps:** Leaflet.js & OpenStreetMap
*   **Icons:** Lucide React
*   **State Management:** React Context API

## 📦 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/evanskaija/dalali.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run in development mode:**
    ```bash
    npm run dev
    ```
4.  **Build for production:**
    ```bash
    npm run build
    ```

## 📐 Design Philosophy

The Dalali App uses a modern **Glassmorphism** design system. This involves heavy use of `backdrop-filter`, subtle gradients, and rounded corners (`--border-radius: 12px`) to create a floating, premium aesthetic that feels like a native mobile app.
